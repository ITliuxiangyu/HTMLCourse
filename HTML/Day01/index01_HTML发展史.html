1. 什么是HTML？ Hyper Text Markup Language （超文本标记语言） 浏览器可以识别的语言 Tim Berners-Lee （1980提出的）， 被称为万维网之父 超文本标记语言（第一版）1993年 HTML2.0 1995年， IETF（互联网工作小组） HTML3.2 1997年1月， w3c推荐标准 （World Wide Web， 万维网联盟） HTML4.0 1997年12月， w3c推荐标准 HTML4.0.1 w3c推荐标准（在HTML4.0做了一些微小改进） XHTML1.0 2000年 w3c推荐标准 （相对于HTML语言来说要求更加严格一点） XHTML1.1 2001年 w3c推荐标准 （部分基于XML （可扩展性标记语言）） XHTML2.0 完全基于XML语言了 HTML5 2014年10月28号 什么是HTML5？ 狭义的说， HTML5就是超文本标记语言第5版 广义的说， HTML5 + CSS + Javascript HTML： 网页的结构 （毛坯房） CSS： Cascading Style Sheets 层叠样式表， 网页的样式 （相当于对房子的装修） Javascript： 网页的行为 （相当于遥控器可以操作投影仪）
